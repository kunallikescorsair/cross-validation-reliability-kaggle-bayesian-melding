# Data

This project uses publicly available completed Kaggle competition datasets:

1. Santander Customer Transaction Prediction
2. Porto Seguro's Safe Driver Prediction
3. Homesite Quote Conversion

Raw Kaggle datasets are not included in this repository due to competition data access terms and file size considerations.

To reproduce the experiments, download the datasets directly from Kaggle and place them under:

- `data/santander/`
- `data/porto/`
- `data/homesite/`

Each folder should contain the original competition files required by the corresponding notebook, such as:

- `train.csv`
- `test.csv`
- `sample_submission.csv`

The repository includes processed experiment outputs and final research datasets under the `results/` directory.
